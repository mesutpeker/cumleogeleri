# cumleogeleri

Türkçe dil bilgisinde **cümlenin öğeleri** konusunu etkileşimli olarak öğreten web uygulaması.

## Canlı Site

https://mesutpeker.github.io/cumleogeleri/

## Özellikler

- Öğe çözümleme
- Kelime grubu bölme
- Pratik ve test modu
- Serbest cümle analizi
- 150 cümlelik veritabanı
- Çalışma kağıdı hazırlama
- Akıllı tahta modu

## Yerel Çalıştırma

```bash
npm install
npm run dev
```

Statik dosyalar doğrudan `index.html` üzerinden de açılabilir.
